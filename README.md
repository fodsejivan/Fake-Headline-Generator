# Fake-Headline-Generator
# 📰 Fake News Headline Generator

A fun and beginner-friendly Python project that generates random fake news headlines using predefined subjects, actions, and places.

---

## 🚀 Features

* 🎲 Random headline generation
* 🔁 Runs continuously until user exits
* 🧠 Uses Python `random` module
* 💻 Simple CLI-based interaction

---

## 🛠️ Technologies Used

* Python
* Random Module

---

## 📂 Project Structure

```
fake-headline-generator/
│── headline.py
│── README.md
```

---

## ▶️ How to Run the Project

1. Make sure Python is installed on your system
2. Clone this repository:

```
git clone https://github.com/fodsejivan/Fake-Headline-Generator.git
```

3. Navigate to the folder:

```
cd Fake-Headline-Generator
```

4. Run the program:

```
python headline.py
```
